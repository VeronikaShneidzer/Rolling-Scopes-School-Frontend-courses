Task: what is represented on the picture?

What is it?

This module is a task. You are offered a picture of the animal and it is necessary to enter what animal is depicts on it.

Where to use?

You can use it wherever you need to use captcha.

How to use?

First create in your site form:
<form>
<div id="task"></div>
<button id="taskButton">Answer</button>
</form>
 Then in index.js you find a function animalsIcons(). Use it in your site.
This function return result.
Add event listener on button. And when user click on button check unswer with result of function.
Example of eventListener you can find in runTask() function in index.js. ATTENTION! You need to use Lodash for this component.
