HP image.

What is it?

This component is an index of hp.

Where to use?

You can use it in your game to show living standard.

How to use?

First create in your site block:
<div id="gameField"></div> Then in index.js you find a function drawTextHP(hp, coordinate, layer). Use it in your site.
hp - is hp of player/enemy,
coordinate - is X coordinate,
layer - is a layer for image.
This function return layer.
ATTENTION! You need to use Konva for this component.
